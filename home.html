Game Info

1. Each player has a total health of 100.
2. If health of one player is 0 then the other one wins the match and the winner gets a point.
3. A tournament consists of five such matches. 4. Whoever the wins
most matches wins the tournament.

Player1

Left Arrow : Move left.
Right Arrow : Move right.
Down Arrow : Shoot.

Player2

Key A : Move left.
Key D : Move right.
Key S : Shoot.
